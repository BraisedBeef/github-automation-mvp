Shared Traffic Package Refund Instructions

Operational Scenario

This chapter guides the user to purchase a shared traffic package. Once purchased, it will take effect immediately and automatically offset the traffic charges generated by the bandwidth of the EIP that is billed on demand (based on traffic), until the traffic package is used up or expires.

Overview

The shared traffic package is a bandwidth traffic plan that is easy to use and cost-effective. Once purchased, it takes effect immediately and automatically offsets the traffic charges generated by the EIP bandwidth billed on demand (based on traffic), until the traffic package is used up or expires.

Three types of shared traffic packages are supported: Dynamic BGP type, which supports offsetting the traffic charges of the dynamically-billed BGP bandwidth; Cat Cloud Premium BGP type, which supports offsetting the traffic charges of the Cat Cloud Premium BGP bandwidth; and Cat Cloud Preferred BGP type, which supports offsetting the traffic charges of the Cat Cloud Preferred BGP bandwidth.

The shared traffic package supports both monthly and annual plans, with the annual plan being more cost-effective. Multiple shared traffic packages can be purchased, and the one that is closer to expiration will be used first.

If the purchased shared traffic package is within its valid period, the charging method is to deduct the purchased shared traffic package’s traffic quota first, and any excess will be billed on demand.

Once the shared traffic package expires, it will not affect the use of your EIP. You only need to ensure that your cloud service account has sufficient balance, and the system will automatically settle the charges on demand.

Shared Traffic Package Usage Restrictions

The shared traffic package is a one-time payment that takes effect immediately and does not support a specified effective date.

Once the shared traffic package is purchased, it cannot be modified or canceled, and renewals are not supported after expiration.

The shared traffic package supports monthly and annual plans, and unused traffic after expiration cannot be used.

The shared traffic package only applies to bandwidth billed on demand (based on traffic) and must distinguish between different dynamic BGP line types.

The shared traffic package does not apply to a specific EIP bandwidth.

The shared traffic package does not apply to shared bandwidth.

When purchasing a shared traffic package, if there are unpaid orders within the payment deadline, you cannot purchase the shared traffic package. Please cancel the unpaid orders or pay the orders before purchasing the shared traffic package.
